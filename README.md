# Предисловие
APanel - Плагин, упрощающий жизнь модераторам на вашем сервере.

**Версия: 1.16.5-1.21.4 (Paper)**

> [!IMPORTANT]
> Плагин требует PlaceholderAPI

# Функционал
Благодаря умной панели, модераторам теперь не придется тратить много времени на блокировку нарушителя,
в конфигурации можно добавить любого рода наказание, и после использовать их, нажав на иконку.
Самый сок в том, что команды будут отправляться от имени того, кто открыл панель, 
Для использования достаточно ввести команду /panel ник_нарушителя (работает даже если нарушитель оффлайн)

# Конфиг
Базовый конфиг представлен ниже
```
panel:
  size: 45
  name: "Караем игрока {player}"
  items:
    ban_1:
      material: IRON_SWORD
      name: "&cKillAura - 7d"
      lore:
        - "&7Бан за читы на 7 дней"
      glow: false
      slot: 20
      command: "tempban {player} 7d читы"

    ban_2:
      material: FEATHER
      name: "&cFly - 7d"
      lore:
        - "&7Бан за читы на 7 дней"
      glow: false
      slot: 22
      command: "tempban {player} 7d флай"

    unban:
      material: TOTEM_OF_UNDYING
      name: "&aРазбан"
      lore:
        - "Разбан игрока"
      glow: false
      slot: 24
      command: "pardon {player}"

    filler:
      material: BLACK_STAINED_GLASS_PANE
      name: " "
      lore: []
      glow: true
      slot: [0-9, 17-18, 26-27, 35-44]

messages:
  noPerm: "&cУ вас нет прав"
  empty: "&cВведите ник игрока"
  reload: "&aПлагин перезагружен"
```

# Возможности
Как Вы уже поняли, в конфиге настраиваются виды наказаний и сообщения в чат. Можете создавать сколько угодно кнопок, ограничение лишь в размере панели.

# Плейсхолдеры
В плагине поддерживаются сторонние плейсхолдеры. 

# Кратко о командах и разрешениях

> ### Команды
**/panel {nick}** - Открыть панель\
**/panelreload** - Перезагрузить плагин.

> ### Разрешения
**apanel.*** - Доступ ко всем командам.\
**apanel.use** - Доступ к /panel ник_нарушителя.\
**apanel.reload** - Доступ к /panelreload.\
